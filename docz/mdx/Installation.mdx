# Installation

### 1. Install the package/s that you want (choose between yarn or npm)

```bash

yarn add @ace/box

```
```bash

npm install -S @ace/box

```

### 2. Use it in your project


```jsx

import React from 'react';
import Box from '@ace/box'; <-- This is the interesting part

function Dropbox() {
  return (
    <Box>Drop<Box>
  );
}

```
