## Default
Inputs rely on you as a developer to provide `value` and handle the
`onChange` event!

```ts

onChange: (event: React.ChangeEvent<HTMLInputElement>) => void

```

## Input with hint

<Playground>
  {
    () => {
      const [value, setValue] = useState('');
      return (
        <Input
          id="username-input-hint"
          label="Username"
          value={value}
          onChange={({ currentTarget: { value } }) => setValue(value)}
          hint="Hint!"
        />
  
      );
    }
  }
</Playground>
